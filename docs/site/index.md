## What makes graceglyph different

- Signal-ready runtime direction for low-latency updates.
- Capability-aware rendering (colors, hyperlinks, graphics protocol support).
- Full app-shell and component primitives instead of prompt-only workflows.
- First-class test toolkit with snapshots, queries, fake timers, and async helpers.
