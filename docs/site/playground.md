# Interactive Playground

The playground replays ANSI output generated by real `graceglyph` renders under
multiple terminal capability profiles.

- `Full`: rich terminal defaults.
- `Dumb`: compatibility fallback profile.
- `Graphics`: inline-image capable profile (kitty/sixel/iTerm2 flags enabled).

Use the selectors to compare behavior changes across capability sets.

<PlaygroundTerminal />

## How it works

- Fixtures are generated by `scripts/generate-playground-fixtures.ts`.
- The generator renders examples with the actual runtime and writes
  `docs/site/.vitepress/playground-fixtures.json`.
- `docs:build` regenerates fixtures so docs stay aligned with runtime behavior.
